# Green eggs and IAM
Welcome to my silly little series where I explore IAM concepts through the lens of beloved Childrens book titles.

Im a security engineer who primarily focuses on Identity and access management. Journey with me as I follow my child like wonder through the process of practicing securing tried and true tools like Active Directory, while mixing in new fresh cloud tools like Microsoft Entra and Okta
